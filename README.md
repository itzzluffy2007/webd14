# Task 14 - JavaScript Loops Examples

This task demonstrates several JavaScript loop-based operations using a single number.

## Files

- `index.html` - Minimal HTML page that loads the JavaScript file.
- `script.js` - Contains the loop logic and prints results to the browser console.

## What it does

For the value `n = 153`, the script performs:

1. Sum of the first `n` natural numbers.
2. Multiplication table for `n` from 1 to 10.
3. Sum of the digits of `n`.
4. Prime number check for `n`.
5. Listing all factors of `n`.
6. Armstrong number check for `n`.

## How to run

1. Open `index.html` in a browser.
2. Open the browser developer console.
3. View the printed results from `script.js`.

## Notes

- The current number used is `153`.
- You can change `n` in `script.js` to run the same checks for a different value.
